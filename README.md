# Bounce Physics
Простая, но увлекательная аркадная игра на iOS: отбивай мячик платформой, набирай очки за каждый отскок, не дай мячу упасть вниз. Реалистичная физика, рекорды, достижения и минималистичный дизайн.

## Demo Video
https://github.com/hxm9q/BallPhysicsGame/releases/tag/v1.0-demo

## Основные требования и особенности проекта

- **Архитектура**: SwiftUI + MVVM с чистым разделением слоёв
- **Разделы приложения** (5 табов):
  1. Home — приветственный экран
  2. Game — основная игровая сцена
  3. Stats — статистика и рекорд
  4. Achievements — список достижений
  5. Settings — настройки
- **Навигация**: TabView + NavigationStack в каждом табе
- **Адаптация под экраны**: Полностью адаптивный интерфейс (GeometryReader, safeAreaInsets, .ignoresSafeArea, без фиксированных размеров)
- **Хранение данных**: UserDefaults (рекорды и достижения)
- **Производительность**: Только async/await, нет блокирующих вызовов на главном потоке
- **Обработка ошибок**: Централизованный AppError + ViewState (idle/loading/loaded/error)
- **Логирование**: Лёгкий logger (без чувствительных данных)
- **Дизайн**: Минималистичный, поддержка светлой/тёмной темы, цвета из Assets
- **Сборка**: Проект собирается без предупреждений и ошибок (Xcode 15+ / iOS 16+)

1. **Иконка приложения**  
   - AppIcon в Assets.xcassets (1024×1024 базовый) и в папке с проектом

2. **Скриншоты**  
   Рекомендуемые размеры (iPhone 6.7"):  
   - Home  
   - Game (во время игры + Game Over алерт)  
   - Stats (рекорд)  
   - Achievements (список)  
   - Settings  

3. **Описание приложения** (для App Store Connect)

**Название:** Bounce Physics  
**Подзаголовок:** Отбивай мячик и бей рекорды!  

**Описание:**  
Bounce Physics — динамичная аркадная игра, где ваша задача — не дать мячику упасть, отбивая его платформой. Реалистичная физика, ускорение мяча, система очков и достижений. Тренируйте реакцию и соревнуйтесь с самим собой!

**Особенности:**
- Простое управление одним пальцем  
- 5 разделов: игра, статистика, достижения, настройки  
- Локальные рекорды и прогресс достижений  
- Полностью адаптивный интерфейс для iPhone
- Минималистичный дизайн  

**Ключевые слова:** bounce ball, paddle bounce, physics game, arkanoid, endless arcade, reflex game  

**Возрастной рейтинг:** 4+  
**Категория:** Игры → Аркады
